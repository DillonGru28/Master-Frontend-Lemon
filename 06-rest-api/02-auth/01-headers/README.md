# 01 Headers

In this example we will login an user using a JWT token in Http Headers and load client and order collections.

# Steps to build it

- `npm install` to install previous sample packages:

```bash
npm install
```

- Start `front` app:

```bash
cd ./front
npm start
```

- Start `back` app:

```bash
cd ./back
npm start
```
